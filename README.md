# Swift Fundamentals & Control Flow

A collection of small Swift problems implemented in a Playground to practice core language constructs and execution flow.

## What Was Practiced
- Arithmetic operations
- Conditionals
- Loops
- Switch statements
- String and integer manipulation

## Learning Focus
Core Swift fundamentals and program flow.

## How This Helped Later
Used in validation logic, UI state handling, and API response processing.
